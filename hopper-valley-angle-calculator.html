Plate Angle, Surface Area & Volume (Industrial Design)

Engineering Formulas

ΔL = (Lt − Lb) / 2
ΔW = (Wt − Wb) / 2

θL = tan⁻¹ ( H / ΔL )
θW = tan⁻¹ ( H / ΔW )
θV = tan⁻¹ ( H / √(ΔL² + ΔW²) )

H = tan(θV) × √(ΔL² + ΔW²)

Volume = (H / 3) × (Atop + Abottom + √(AtopAbottom))
