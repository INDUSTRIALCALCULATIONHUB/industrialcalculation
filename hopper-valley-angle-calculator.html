Plate Angle & Valley Angle (Industrial Design)

Engineering Formulas

Geometric parameters:

ΔL = (Lt − Lb) / 2
ΔW = (Wt − Wb) / 2

Plate angles:

θL = tan⁻¹ ( H / ΔL )
θW = tan⁻¹ ( H / ΔW )

Valley angle:

θV = tan⁻¹ ( H / √(ΔL² + ΔW²) )

Mode 2 rearrangement:
H = tan(θV) × √(ΔL² + ΔW²)
